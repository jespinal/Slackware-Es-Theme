== Description ==

Slackware-Es is a lightweight WordPress theme showcasing Slackware Linux-related content. It's intended as a simple classic-theme, minimal dependencies, and easy customization.

== Installation ==

1. Upload the `Slackware-Es` folder to the `/wp-content/themes/` directory.
2. Activate the theme through the 'Appearance' -> 'Themes' screen in WordPress.
